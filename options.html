Display Options

Feature Options

Enable or disable specific features:

You can toggle which features appear in the selection menu and right-click context menu.

Custom Commands

Create your own custom commands:

Add custom commands that will appear in both the selection menu and right-click context menu. When used, they will format the text as: "selected text", [your custom command]

Language Settings

How to use language translation:

  1. Select your preferred language from the dropdown below
  2. Click the "Save Settings" button
  3. When you select text on any webpage, a "Translate to [Your Language]" button will appear
  4. Click this button to send the text with translation instructions
